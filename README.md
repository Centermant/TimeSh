Система учета времени сотрудников

Простое веб-приложение для отметки времени прихода/ухода сотрудников с возможностью администрирования.

Функционал:

    Отметка времени: Сотрудники могут отмечать "приход" и "уход" на рабочем месте.
    Несколько должностей: Сотрудник может иметь несколько должностей, отметки делаются для конкретной должности.
    Администрирование: Администратор может управлять сотрудниками, подразделениями, должностями и просматривать отчеты.
    Выгрузка данных: API для получения данных о сотрудниках и их рабочем времени в формате JSON.

Технологии:

    Backend: Node.js, Fastify, PostgreSQL
    Frontend: HTML, CSS, JavaScript
    Контейнеризация: Docker, Docker Compose

Запуск:

    Убедитесь, что установлен Docker.
    Соберите и запустите проект: docker-compose -f 'docker-compose.local.yml' up -d --build
    Приложение будет доступно по адресу http://localhost:3000.
    Adminer (для просмотра БД) будет доступен по адресу http://localhost:8080.

Настройка:

    Строка подключения к БД и JWT-секрет указываются в файле backend/.env.local.
